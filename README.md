# share APP简介

软件环境：Windows+java+MySQL
编程工具：Android Studio，Eclipse，Navicat for MySQL
项目描述：本应用是一款专注插画分享的美学平台社交型APP，服务于插画爱好者与专业插画师，主要功能有每日精选（从用户上传作品筛选出精选作品）可供户快速查看；圈子可进行每日打卡与作品交流，用户也可通过发布分享自己的作品；可以收藏、评论、关注可收到消息通知
